# React-Mobx-SSR-Template

基于 typescript 的的全栈同构模板

## 技术选型

- typescript
- nestjs
- eslint
- react
- postcss
- mobx
- typeorm
- pm2
- hbs
- rxjs
- mongoose
- axios
- jest
- swagger
- styled-components

## 特性

- 全栈热加载热更新
- 同构应用，服务端挂掉不影响客户端单独使用
- 简单的税局脱水注水
- 安全易用的按需加载
- css公共提取与热加载
