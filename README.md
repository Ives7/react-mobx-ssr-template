# React-Mobx-SSR-Template

基于 typescript 的的全栈同构模板

## 技术选型

- typescript
- nestjs
- eslint
- react
- postcss
- mobx
- typeorm
- pm2
- hbs
- rxjs
- mongoose
- axios
- jest
- swagger
- styled-components

## 特性

- 全栈热加载热更新
- 同构应用，服务端挂掉不影响客户端单独使用
- 数据的脱水注水的管理与解决方案
- css 公共提取热加载

## 研发中特性

- 自动external三方应用，自动注入到HTML
- 完善的异步解决方案
